# 1.0.0

首次发布。

- 支持搜索美股/港股股票，按名称或代码（如 AAPL、00700）添加自选
- 持仓卡片展示：现价、涨跌幅、持仓成本、市值、盈亏
- 展开卡片查看详情：持仓均价、加仓/减仓次数、最高/最低买入价
- 资产汇总卡片：总资产、总市值、总盈亏、已实现盈亏、总股息、股息率、持仓比例
- 加仓、减仓、平仓、派息操作记录
- 派息管理：记录每股派息、税率、日期，自动计算税后股息
- 实时行情：腾讯（港股）+ 东方财富（美股）双数据源
- 实时汇率：USD/CNY/HKD 自动更新
- 多币种支持：默认货币可在 USD/CNY/HKD 间切换
- iCloud 多设备数据同步
- 设置页：货币、排序方式、同步开关、平仓保留开关
- 股票 Logo 缓存加速显示
- 自定义 App 图标
- App 版本号自动获取，不再硬编码
- 优化 App 图标和界面显示
- 修复刷新后持仓未上传 iCloud，导致重进 App 丢失数据的问题
- 修复部分特殊股票行情获取失败的问题
- 优化 iCloud 同步逻辑，提升数据同步稳定性
