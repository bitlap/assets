# 1.2.0

### UI 视觉统一
- **配色统一**：底部 tab 图标不分选中/未选始终有颜色（股票蓝 `#5B9CF6` / 资产橙 `#FF9F0A`），icon 在上文字在下
- **字体层级**：第一行左右统一 `fontSize: 13`，第二行左右统一 `fontSize: 12`，所有计算公式/时间/日期统一 `fontSize: 11, color: Colors.grey[500]`
- **分割线统一**：所有 Divider 统一 `thickness: 0.5, color: Color(0xFF1C1C1E)`
- **按钮风格**：悬浮按钮和齿轮改为白底黑图标；记录 button icon 蓝色 `#5B9CF6`，更多 button icon 橙色 `#FF9F0A`
- **分类标题**：字号从 16 改为 14，与资产内容标题一致
- **Switch**：开启态改为蓝色 `#5B9CF6` 轨道和拇指

### 股票页面优化
- **盈亏列**：通过 `Spacer()` 推到最右侧，header 右 padding 改为 26 匹配卡片边缘
- **卡片主行**：改为 `Expanded(flex:2)+Expanded(flex:2)+Spacer+profit` 与表头对齐
- **持仓列**：`Alignment.center` → `Alignment.centerLeft` 解决 iPad 宽屏偏移
- **股息/操作记录**：股息记录第三行改为 Row（左侧操作时间 + 右侧派息日期）；操作记录第三行改为 Row（左侧更新时间 + 右侧创建时间），Flexible + ellipsis 防溢出
- **计算公式**：去掉「股」「$」「每股」单位符号，纯数字显示
- **弹窗**：股票更多弹窗选项间加 Divider

### 资产页面优化
- **icon/颜色**：理财/基金 icon `#2C2C2E` → `#5B9CF6`；名称文字 `#8E8E93` → `Colors.white`
- **弹窗**：资产添加弹窗选项间加 Divider

### 新增资产类型
- **活期存款**：新增 `AssetType.current` 类型，`CurrentAccount` 模型（同 CashAccount 结构），icon `Icons.account_balance` / `#5AC8FA`
- **公积金**：新增 `AssetType.providentFund` 类型，`ProvidentFundAccount` 模型（同 CashAccount 结构），icon `Icons.home_work` / `#AF52DE`
- **通用弹窗**：`_BalanceDialog` 通用弹窗供现金/活期/公积金复用
- **数据驱动**：`showAddAssetSheet` 改为数据驱动 `_assetOptions` 列表，新增类型只需加一项
- **文案更新**：现金提示从「例：活期账户、钱包」改为「例：钱包、储蓄卡」

### 数字格式化调整
- **formatCompact**：`≥ 1亿` → `X.XX亿`，`≥ 100万` → `X.XX万`，`< 100万` → 显示完整数字

### 刷新逻辑重构
- **统一入口**：`_refreshAll()` 统一负责先加载 iCloud 再获取行情
- **清理**：删除 `_isLoading`、加载覆盖层、冗余 `_fetchExchangeRatesWithoutRebuild`

### 修复
- 修复资产列表名称颜色显示异常
- 修复理财/基金 icon 颜色显示错误
